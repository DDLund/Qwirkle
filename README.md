# Qwirkle
This repository gives matlab code to compute the exact probability of having a starting hand of a given score. This confirms theoretical results reported in the paper below.

> [Probabilities of Qwirkle Hand Values](https://www.tandfonline.com/doi/abs/10.1080/07468342.2018.1490592?journalCode=ucmj20), The College Mathematics Journal, 2018
